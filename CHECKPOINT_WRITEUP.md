# Checkpoint Writeup

## Proposition
**"Global fossil fuel subsidies are declining."**

---

## Page 1: Visualizations Supporting "Subsidies ARE Declining"

**Visualization:** `checkpoint_viz1_declining.png`

This visualization argues that fossil fuel subsidies are declining by:
- Showing US subsidies dropped 12% from $860B (2019) to $757B (2022)
- Highlighting 5 "leading nations" (US, Canada, France, Brazil, Estonia) with declining subsidies as % of GDP

---

## Page 2: Visualizations Supporting "Subsidies are NOT Declining"

**Visualization:** `checkpoint_viz2_increasing.png`

This visualization argues that fossil fuel subsidies are NOT declining by:
- Showing top 5 emitters all increased subsidies: India (+93%), China (+85%), Japan (+55%), Russia (+25%), USA (+4%)
- Displaying environmental/health costs growing from $2.9T (2015) to $4.1T (2022) to $5.7T projected (2030)

---

## Which Set Am I Leaning Toward?

I am leaning toward the **"Subsidies are NOT Declining"** visualization because it presents a more complete and honest picture of global trends. The "declining" visualization requires cherry-picking specific countries and time periods to support its narrative.

---

## Deceptive Techniques Used

### "Subsidies ARE Declining" Visualization (Page 1)

| Technique | Description |
|-----------|-------------|
| **Truncated Y-axis** | Y-axis starts at $700B instead of $0, making the 12% decline appear much more dramatic visually |
| **Cherry-picked countries** | Only shows 5 countries with declining trends, ignoring 136 countries where subsidies increased |
| **Selective time period** | Uses 2019-2023 (peak to relative low) instead of showing full 2015-2023 data where subsidies actually increased |
| **Positive framing** | Title "Progress on Climate" and "Leading Nations" implies global success that doesn't exist |
| **Metric selection** | Uses % of GDP which shows smaller numbers; absolute values tell a different story |

### "Subsidies are NOT Declining" Visualization (Page 2)

| Technique | Description |
|-----------|-------------|
| **Alarming title** | "Growing Global Crisis" primes viewer toward alarm (though data-supported) |
| **Red color scheme** | Uses danger-associated colors to emphasize severity |
| **Including projections** | Shows 2030 projections which are uncertain estimates |
| **Absolute USD values** | Uses trillions instead of % of GDP, resulting in larger, more alarming numbers |

---

## Summary

The same dataset can tell completely opposite stories depending on visualization choices. The "declining" narrative requires more deceptive techniques (truncated axes, cherry-picking) to sustain, while the "not declining" visualization presents global data more honestly but still makes editorial choices in framing.
